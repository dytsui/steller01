# Directory guide

- `app/` product routes and API handlers
- `components/` premium UI building blocks
- `lib/analysis/` trusted analysis engines
- `lib/browser/` client-side diagnostics and local persistence
- `lib/server/` demo data and server helpers
- `workers/` Cloudflare queue / Durable Object / proxy worker examples
- `db/` D1 schema and seed
- `data/` issue catalog, standards, demo records, i18n copy
- `public/brand/` logo assets
- `styles/` design system CSS
- `docs/` deployment and architecture notes
