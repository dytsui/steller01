# Directory snapshot

This redo package is intentionally thicker than a lightweight demo.

## Major folders

- `app/` pages and API routes
- `components/` UI and product modules
- `lib/analysis/` trusted analysis layer
- `lib/server/` demo repositories, Gemini, payload helpers
- `public/brand/` logo assets
- `public/demo/screens/` visual mock assets
- `db/` D1 schema and seed
- `workers/` queue and Durable Object scaffolding
- `docs/` deploy, architecture, UX, Screen Mode, queue notes

## Root deployment
The repository remains root-deployable and non-monorepo.
