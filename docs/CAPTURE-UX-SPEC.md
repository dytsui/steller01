# Capture UX spec

## Goal
Keep the video stage visually dominant.

## Layout targets
- mobile: ~78dvh video stage
- desktop: ~80dvh max-height feel
- overlays live on top of the video, not below it

## Visible overlays
- skeleton
- body axis
- spine line
- shoulder line
- hip line
- head stability box
- target / reference line
- ghost wrist path
- phase label
- score / tempo HUD
- previous vs current / pro compare ribbons

## Bottom controls
- preview start/stop
- record start/stop
- play / pause
- previous keyframe / next keyframe
- slow motion
- compare
