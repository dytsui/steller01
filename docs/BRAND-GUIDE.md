# Brand guide

## Positioning
steller01 should feel like:
- elite sports technology
- calm coaching authority
- premium mobile-first interface
- evidence before opinion

## Brand line
**Trusted analysis first. Trusted coaching report second.**

## Palette
- Deep Field: `#06110f`
- Tour Mint: `#55d6b2`
- Champagne Gold: `#d2a647`
- Mist White: `#f2f7f5`

## Usage
- Deep field is the dominant background.
- Mint signals trusted system state and actionable feedback.
- Gold signals premium benchmark / pro comparison / membership surfaces.
- Keep long copy calm and minimal. Let the video area dominate the page.
